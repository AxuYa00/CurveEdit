# CURVEEDIT - 交互式曲线编辑器

CURVEEDIT 是一个基于 Python 与 Cursor 框架开发的交互式曲线编辑软件。用户可以通过鼠标点击在画布上创建顶点，系统将自动绘制一条光滑连续的曲线。

## 主要功能

- 交互式顶点绘制
- 顶点编辑与拖动
- 曲线形态调控
- 单窗口交互界面

## 安装要求

- Python 3.x
- Cursor SDK
- NumPy

## 安装步骤

1. 克隆仓库
2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 使用方法

运行主程序：
```bash
python main.py
```

## 操作说明

- 左键点击：添加顶点
- 拖动顶点：调整曲线形状
- 使用右侧控制面板调整曲线参数
